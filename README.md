# Smart CRM Bot

Telegram-бот для управления личными контактами и нетворкинга.

## Возможности
- 🔍 Умный поиск контактов по описанию сферы/интересов
- ⏰ Напоминания о тех, с кем давно не общались
- 💡 Генерация фраз для начала диалога (ледоколов)
- 🤖 Автотегирование контактов на основе переписки
- 📊 Веб-панель с аналитикой и дашбордом
