# Внесение вклада

Спасибо, что решили помочь проекту! Вот несколько рекомендаций.

## Как сообщить об ошибке
1. Проверьте, не описана ли ошибка ранее в [Issues](https://github.com/lo-van/smart-crm-bot/issues).
2. Если нет, создайте новый issue, описав:
   - Ожидаемое поведение.
   - Фактическое поведение.
   - Шаги для воспроизведения.
   - Версии Python, ОС и зависимостей.

## Как предложить улучшение
Откройте issue с меткой `enhancement` или сразу сделайте Pull Request.

## Процесс разработки
1. Форкните репозиторий.
2. Создайте ветку для вашей фичи (`git checkout -b feature/amazing-feature`).
3. Внесите изменения и закоммитьте (`git commit -m 'Add amazing feature'`).
4. Запушьте ветку (`git push origin feature/amazing-feature`).
5. Откройте Pull Request.

## Стиль кода
- Следуйте PEP 8.
- Используйте осмысленные имена переменных.
- Добавляйте комментарии для сложных участков.

Спасибо за ваш вклад!
